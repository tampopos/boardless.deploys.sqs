# boardless.deploys.sqs

sqs の展開

## Required

- .Net core 2.2
- node.js

#### Options

- vscode

## Usage

#### 始めに

`npm install`

#### ビルド

`npm run build`

#### テスト

`npm run test`

#### 展開

`npm start`

#### Debug

vscode でデバッグ実行可能
